# 墨舟 InkNote v1.5

> 一款轻量、离线可用的 Markdown 学习助手 — 单 HTML 文件，零依赖，打开即用。

---

## 快速开始

直接下载 [InkNote.html](InkNote.html) 用浏览器打开即可，无需安装任何东西。

支持的浏览器：Chrome / Edge / Firefox（推荐最新版）

---

## 功能

| 功能 | 说明 |
|------|------|
| 文件夹导入 | 拖入整个笔记目录，自动生成文件树 |
| 全文搜索 | 多关键词、正则表达式搜索（Ctrl+K） |
| **AI 助手** | **浮动按钮 + 可拖拽聊天面板，支持流式对话，配置加密存储（Ctrl+Shift+A）** |
| 学习进度 | 右侧面板标记每章学习状态，自动复习提醒 |
| 目录导航 | 侧边栏自动生成目录树，滚动同步高亮当前章节 |
| 原地编辑 | 分屏实时预览，上下/左右布局可切换，支持粘贴图片转 base64（Ctrl+E） |
| 快捷键 | Ctrl+K/O/H/E/S + Esc 按层级关闭浮层 |
| 实时分享 | 通过 PeerJS 与他人实时同步文件，支持 AES 端到端加密 |
| 代码高亮 | 50+ 种语言语法高亮（Prism.js） |
| 三态主题 | 暗色 / 亮色 / 跟随系统，默认跟随系统 |
| 数据持久化 | 笔记自动保存到 IndexedDB，刷新不丢失 |
| 历史记录 | 点击 📚 按钮查看和恢复之前的笔记 |
| 一键清理 | 历史记录支持单条删除和全部清空 |
| 上次打开 | 拖拽区显示「上次打开」快捷入口，一键恢复 |
| 侧边栏过滤 | 文件树顶部输入框实时过滤文件名 |
| 阅读统计 | 总阅读字数、学习天数、连续打卡 |
| 导出分享 | 导出为 PDF（打印）、Word 或独立 HTML 文件 |
| 自定义语法 | 注册自定义正则 Markdown 扩展规则（插件系统） |
| 示例文档 | 首次打开自动加载示例笔记，零配置即可体验 |
| 回到顶部 | 右下角浮动按钮，ease-out-cubic 平滑滚动动画 |
| PWA 支持 | HTTPS 环境下可安装到桌面，支持离线缓存 |

## 使用示例

```
1. 打开 InkNote.html
2. 拖入你的笔记文件夹（或点击「导入」按钮选择文件夹）
3. 侧边栏自动生成文件树，点击文件浏览
4. 右侧「学习进度」面板标记章节完成状态
5. 点击 ✎ 或按 Ctrl+E 进入编辑模式，分屏实时预览
6. Ctrl+S 保存，粘贴图片自动转 base64
7. 点击 🔗 分享，生成房间号，对方输入即可实时同步
```

## 技术栈

全是纯前端原生技术，无框架依赖：

- 语言：HTML + CSS + JavaScript（~4450 行，单文件）
- **存储**：IndexedDB（文件持久化）+ localStorage（配置/进度/阅读统计/自定义规则）
- **外置依赖（CDN）**：Prism.js 代码高亮、PeerJS P2P 分享
- **离线能力**：首次加载后完全离线可用，所有数据存本地；PWA 安装到桌面

详见 [墨舟InkNote-架构文档1.2.md](墨舟InkNote-架构文档1.2.md) 了解模块划分、数据流和存储策略。

### v1.5（当前）

新增 AI 助手 + 可拖拽聊天面板，修复移动端定位问题：

- **新增 AI 浮动按钮** — 右下角灵眸 FAB，支持拖拽，桌面端 90px / 移动端 70px 底部定位
- **新增 AI 可拖拽聊天面板** — 桌面端右上角固定 (top: header+20px, right:20px)，支持拖拽移动
- **移动端 AI 面板底部抽屉适配** — ≤640px 转为底部抽屉 (bottom:8px, left/right:8px, top:auto)，≤480px 进一步收窄边距至 6px

- **亮暗主题兼容** — 所有新增 CSS 纳入 `:root` 和 `[data-theme="light"]` 变量体系

### v1.4

修复 1 个异常场景和删除 1 个功能遮罩

- **修复移动端进度面板 toggle 按钮无响应** — 点击「≡」按钮后进度面板打开又被立即关闭（事件冒泡导致 document click handler 干扰），添加 stopPropagation() 阻止冒泡
- **修复移动端进度面板初始化状态** — 页面加载后 toggle 按钮正确显示，进度面板默认隐藏

---

### v1.3

修复 2 个异常场景，新增 2 个体验优化：

- **进度导出按钮点击后面板关闭** — 事件冒泡导致面板消失，按钮添加 `stopPropagation`
- **导出/下载偶发静默失败** — `URL.revokeObjectURL()` 过早释放，延迟 200ms 后回收
- **侧边栏 active 切换缓存** — `_activeTreeItem` DOM 引用缓存，减少 `querySelectorAll` 遍历
- **自定义平滑滚动** — `requestAnimationFrame` + `ease-out-cubic` 替代原生 `behavior: smooth`

---

### v1.2

- **新建笔记编辑器未激活** — `createNewFile()` 跳过预览直接进入编辑模式
- **保存流程优化** — `showSaveFilePicker` 一步选目录改文件名，不支持的浏览器走 fallback
- **关闭文件侧边栏未清理** — 关闭未保存文件时从侧边栏移除
