# 墨舟 InkNote

> 一款轻量、离线可用的 Markdown 学习助手 — 单 HTML 文件，零依赖，打开即用。

---

## 快速开始

直接下载 [InkNote.html](InkNote.html) 用浏览器打开即可，无需安装任何东西。

支持的浏览器：Chrome / Edge / Firefox（推荐最新版）

---

## 功能

| 功能 | 说明 |
|------|------|
| **文件夹导入** | 拖入整个笔记目录，自动生成文件树 |
| **全文搜索** | 多关键词、正则表达式搜索 |
| **学习进度** | 右侧面板标记每章学习状态 |
| **复习提醒** | 标记完成后自动设置复习周期 |
| **原地编辑** | 点击 ✎ 按钮直接编辑笔记 |
| **实时分享** | 通过 PeerJS 与他人分享文件，支持连接管理 |
| **代码高亮** | 50+ 种语言语法高亮（Prism.js） |
| **暗色/亮色** | 右上角一键切换主题 |
| **数据持久化** | 笔记自动保存到 IndexedDB，刷新不丢失 |
| **历史记录** | 点击 📚 按钮查看和恢复之前的笔记 |
| **一键清理** | 历史记录支持单条删除和全部清空 |

---

## 使用示例

```
1. 打开 InkNote.html
2. 拖入你的笔记文件夹（或点击「导入」按钮选择文件夹）
3. 侧边栏自动生成文件树，点击文件浏览
4. 右侧「学习进度」面板标记章节完成状态
5. 点击 ✎ 进入编辑模式，修改后自动保存
6. 点击 🔗 分享，生成房间号，对方输入即可实时同步
```

## 技术栈

全是纯前端原生技术，无框架依赖：

- **语言**：HTML + CSS + JavaScript（~2900 行，单文件）
- **存储**：IndexedDB（文件持久化）+ localStorage（配置/进度）
- **外置依赖（CDN）**：Prism.js 代码高亮、PeerJS P2P 分享
- **离线能力**：首次加载后完全离线可用，所有数据存本地

详见 [墨舟InkNote-架构文档.md](墨舟InkNote-架构文档.md) 了解模块划分、数据流和存储策略。
